Задание
-

Необходимо для каждой mcc группы вывести 3 наиболее дорогих транзакции за 2019 и 2020 год. 
Если в рамках какой-то mcc группы за год было менее 3 транзакций, то недостающие места должны быть дополнены null.

В результате необходимо вывести столбцы:
- group_name - имя группы
- year - год
- rn - ранг
- transaction_value - сумма транзакции

Результат должен быть отсортирован по имени группы, рангу и году по возрастанию всех полей. 
Все суммы округлите до 2 знака после запятой.

_Пример ответа:_

| Группа | Год | № | Сумма |
|---|---|---|---|
| АЗС | 2019 | 1	| null |
| АЗС | 2020 | 1	| 1200.00 |
| АЗС | 2019 | 2	| null |
| АЗС | 2020	| 2	| 300.00 |
| АЗС | 2019	| 3	| null |
| АЗС | 2020	| 3	| 200.00 |
| АЗС | 2019	| 1	| 300.00 |
| АЗС | 2020	| 1	| 300.00 |
| АЗС | 2019	| 2	| null |
| АЗС | 2020	| 2	| null |
| АЗС | 2019	| 3	| null |
| АЗС | 2020	| 3	| null |
