## Задание 

Мы давно хотим собрать для себя checkup-дашборд. Если коротко - хочу, чтобы у нас был дашборд, на котором будут отображены все основные показатели нашего бизнеса (в данном случае я про IT Resume). Пока что дашборд делать не нужно - давайте начнем с построения визуализаций и их настройки.

Я для себя выделяю такие метрики:

- Воронка "Сделал попытку решить задачу - Решил задачу успешно - Пополнил кошелек"
- Распределение первых и повторных покупок - сколько человек купило первый раз, а сколько совершают уже не первую покупку
- Rolling retention по заходам на платформу с разбивкой по когортам

Очень важно, чтобы везде был удобный фильтр по дате - чтобы мы могли ежедневно ставить удобный для нас период и смотреть статистику за разное время.

Чуть позже тогда вернусь с информацией по построению дашборда.
